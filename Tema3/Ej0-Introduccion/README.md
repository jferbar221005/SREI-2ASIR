# Introduccion a Docker

# Conceptos basicos

Docker es una plataforma que permite empaquetar, distribuir y ejecutar aplicaciones dentro de entornos aislados, ligeros y portátiles. Al incluir el código, las herramientas y las dependencias en un solo paquete, asegura que la aplicación funcione de forma idéntica en cualquier máquina, servidor o entorno en la nube.

# Imagenes Docker y contenedores

Una imagen es una plantilla de solo lectura que contiene todo lo necesario para ejecutar un software (código, bibliotecas y variables). Un contenedor es simplemente una instancia en ejecución de esa imagen.

# Volumenes de almacenamiento

Si un contenedor se borra o se destruye, los datos creados en su interior desaparecen con él. Los volúmenes son el mecanismo que ofrece Docker para guardar esa información de forma persistente y segura fuera del sistema de archivos del contenedor.
